Toggle 2


Pure CSS togglers based on checkbox hack.
This approach introduces bad semantics and is not generally recommended.
But it overcomes some drawbacks of using :target. Its pros are:

    
	
  • multiple active (open) blocks simultaneously
	
  • nested togglable blocks
	
  • styling of active label
	
  • on/off behaviour of the same label
	
  • page would not scroll
	
  • no need for javascipt enhancements

Javascript enhancements



    
	
  • Dropdown — toggle, escape
	
  • Tree — multiple, unscroll, toggle, remember, .accordion
	
  • Toggle — multiple, unscroll, toggle, remember
	
  • Popup — multiple, unscroll, toggle, escape
	
  • Tabs — multiple, unscroll, remember
	
  • Dialog — escape
	
  • Gallery — previous, escape
	
  • Output HTML code (pre[data-src])
	
  • Link or button with confirmation (.confirm[title])
	
  • Link with prompt (a.prompt[title][data-default])
	
  • Check group of boxes (input[data-group], [class~="data-group"])
	
  • Align table columns (table.r5)
	
  • Show dropped uploaded image (input.drop)

Alphabetical





    

  1. .act — active link

  2. .bg — light background

  3. .bg-e — nominal "red" background

  4. .bg-i — nominal "blue" background 

  5. .bg-n — nominal "gray" background

  6. .bg-w — nominal "yellow" background

  7. .bg-y — nominal "green" background

  8. .bord — add border

  9. .btn — button-like link

  10. .c — centrify content inside block

  11. .center — centrify block itself

  12. .close — link to close togglable block

  13. .col-0 — grid cell with non-expanding width

  14. .col-2 — grid cell with expanding width (x2)

  15. .col-3 — grid cell with expanding width (x3)

  16. .dlg — card-like block

  17. .drop — file input to show dragged image

  18. .flip — responsive component, stacked view on mobile

  19. .full — fullscreen block

  20. .gal — fullscreen gallery

  21. .hide — hidden

  22. .hover — highlight link or table row on hover

  23. .icon — small image

  24. .js-control — visibility is controlled by Javascript, hidden by default

  25. .js-show — shown with Javascript

  26. .l — align content to the left

  27. .mar — add vertical margins

  28. .narrow — narrow block width

  29. .nav — horizontal dropdown navigation

  30. .pad — add padding

  31. .pop — popup component wrapper

  32. .print — show only on print (not on screen)

  33. .r — align content to the right

  34. .rad — rounded corners

  35. .read — block width for comfortable reading

  36. .roll — horizontal scrolling of contents

  37. .row — row of grid

  38. .screen — show only on screen (not on print)

  39. .shift — shift item of .row or .nav to the right

  40. .show-desktop — force visibility on desktop (overrides .hide)

  41. .show-mobile — force visibility on mobile (overrides .hide)

  42. .small — small font size

  43. .stick — stick block to the top

  44. .switch — input controlling visibility of following block

  45. .tabs — tabs component

  46. .text-e — nominal "red" text color

  47. .text-i — nominal "blue" text color

  48. .text-n — nominal "gray" text color

  49. .text-w — nominal "yellow" text color

  50. .text-y — nominal "green" text color

  51. .wrap — reasonable page width limit

  52. .yel — temporary highlight targeted block
